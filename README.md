# HLox

## Data Types

1. Number, double-precision floating point

    `1`, `1.2`

2. String

    `"this is a string"`, `"newline\nescape"`

3. Boolean

    `true` `false`

## Expression

Usual stuff. Literals, grouping `(expression)`, unary `- !`, binary operator `+ - * / < <= > >= == !=`. Strict typed.

## Statement

`EXPR ;`
`print EXPR;`
`var NAME (= EXPR)?;`
